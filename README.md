# 🛡️ Safe Tunnel

مدیریت‌کننده حرفه‌ای **Reverse TCP Tunnel** برای اتصال سرور ایران و خارج، با رابط کاربری ساده، نصب سریع و امکانات حرفه‌ای.

## ✨ Features

- Reverse TCP Tunnel
- Multi-Slot (1–10)
- AutoSync
- Health Check
- BBR Optimization
- Multiple Port Forward Methods
- systemd Support

## 🚀 Installation

```bash
bash <(curl -Ls https://raw.githubusercontent.com/Safe-Net5/Safe-Tunnel/main/install.sh)
```

اجرای منوی مدیریت:

```bash
sudo Safe-tunnel
```

📖 راهنمای کامل نصب و استفاده در مستندات پروژه موجود است.

⭐ اگر این پروژه برایتان مفید بود، با دادن **Star** از آن حمایت کنید.
